Install NEP/RIZE

Installation in Windows

In order to start using NEP in Windows you need basically to donwload and install:

  • Python 2.7 for x86
  • The latest version of NEP

Installing Python

Before installing NEP in Windows you need to install Python 2.7 for architecture x86. We recomend to use the Windows x86 MSI Installer from offical Python page.

Click here to redirect to the official Python webpage

Very important

During the Python installation, include the Python directory path in the PATH environment variable. For this, select the Will be installed on local hard drive option, and then click Next (See the figure bellow).

Even when NEP can work with other Python distributions such as Anaconda, the official version from the Python webpage has better support for some robots, such as NAO and Pepper.

Option 1: NEP from installer (user)

  • Download the latest NEP end-user wizard installer from:

Click here to download the current experimental version

Very important

During NEP installation it is needed to have the PC connected to internet. This is due that the installer will download and install some essential Python packages

  • Follow the installation wizard.
  • A black window will appears to install all the needed python packages (using pip) and to build the NEP library. Please wait until the black window is closed automatically.

Option 2: NEP From Github (developer)

Clone or download the most recent version of NEP from:

https://github.com/enriquecoronadozu/NEP

Then run the install.bat file (with double click).

Installation in Ubuntu

  • Clone or download the most recent version of the NEP from:

https://github.com/enriquecoronadozu/NEP

  • Go insode the NEP or NEP-master folder using the linux console and run the install.sh.

  • See the common problems section bellow if you have some installation problems.

Installation in Mac

  • If you will use a NAO or Pepper robot, you need to install and use the newest version of Python 2.7 from the official Python webpage:

https://www.python.org/downloads/

Python distributions and NaoSDK

In Mac the NAO and Pepper SDK do not work with the default Python installation in MacOSX or with Anaconda, please use the Python version from the official python webpage

  • Download the most recent version of NEP from:

https://github.com/enriquecoronadozu/NEP

  • Go insode the NEP or NEP-master folder using the linux console and run the install.sh

  • See the common problems section bellow if you have some installation problems

Common installation problems

Problem running the install.sh script (MacOS and Ubuntu)?

Use the chmod +x install.sh commands to allows the execution of the script and then run the script using ./install.sh
