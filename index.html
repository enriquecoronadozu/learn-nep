What is NEP?

Node primitives (NEP) is an open source programming platform which allows the creation and design of social interactions with robots.

  • Works in Linux, Windows and OSX.
  • Enables end-user and hard-code programming.
  • Use ZeroMQ, nanomsg, normal sockets or ROS for inter-process comunication.
  • Easy to install and use.
  • Mostly written in Python.

The core API of NEP is very lightweight (less that 3Mb). The main requirement is to have a Python distribution installed.

Instalation in Windows

  • Download and install Python 2.7. We recommend the Python (x,y) distribution:

    https://python-xy.github.io/

  • Download the most recent version of the NEP programming platform from:

    https://github.com/enriquecoronadozu/NEP

  • Unzip the folder and run the file install.bat (with double click) that is inside the unzipped folder.

Instalation in Ubuntu

  • Download the most recent version of the NEP programming platform from:

    https://github.com/enriquecoronadozu/NEP

  • Unzip the folder and in linux console run the install.sh script that is inside the unzipped folder.

  • See the “Common problems” section bellow if you have some installation problem.

Instalation in Mac

  • If you will use a NAO or Pepper robot, you need to install and use the newest version of Python 2.7 from:

    https://www.python.org/downloads/

Python distributions and NAOSDK

In Mac the NAO and Pepper SDK do not work with the default Python installation or with Anaconda

  • Download the most recent version of the NEP programming platform from:

    https://github.com/enriquecoronadozu/NEP

  • Unzip the folder and with Mac Terminal run the install.sh script that is inside the unzipped folder.

  • See the “Common problems” section bellow if you have some installation problem.

A dummy example

In order to test that NEP was suceffully installed we will create a simple publisher-subcriber application. For this we need to create 2 python files.

The first one will be a node with a publisher. This script can be denoted as pub.py. This script must to have the next code:

import nep
import time

node = nep.node("pub")
conf = node.conf_pub(network = "direct")
pub = node.new_pub("test", conf)
while True:
    msg = "hello world"
    pub.send_string(msg)
    time.sleep(1)

The second one will be a node with a subscriber. This script can be denoted as sub.py. This script must to have the next code:

import nep

node = nep.node("sub")
conf = node.conf_sub(network = "direct")
sub = node.new_sub("test", conf)
while True:
    s, msg = sub.listen_string()
    if s:
        print msg

Then, execute both scripts in different terminal/console windows. If the message of hello world appears in the console of the subscriber node, then the installation has been successful.
